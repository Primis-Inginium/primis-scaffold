# 🤖 Primis Scaffold

Standardized Next.js template for **Primis-Inginium** applications.

## 🏗 Modular Architecture
This project follows a strict **Core-Feature** modular pattern:
- `@core/*` (`src/core/`): Generic, reusable code.
- `@features/*` (`src/features/`): Domain-specific encapsulated logic.

## 🛡️ AI Governance
Inherits all organization-level governance from the [.github](https://github.com/Primis-Inginium/.github) repository.
- Use `ai_task.yml` for agent tasks.
- Ensure all PRs include the mandatory audit log.

## 🚀 Getting Started
1. `npm install`
2. `npm run dev`
